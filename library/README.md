# Tmazing GitHub Library

This folder is for session exports that you intentionally upload to GitHub.

The app cannot silently save private study files into GitHub from GitHub Pages. At the end of a session, use **End session** in the app to download:

- a local Markdown session file
- a JSON GitHub-library pack
- a Markdown GitHub-library pack

Then upload the files you want to keep here.

Suggested folder naming:

- `library/E104/block-1/`
- `library/E104/block-2/`
- `library/K102/unit-3/`
- `library/session-exports/`

Do not upload private or copyrighted course materials unless you are allowed to store them in a public GitHub repository.
